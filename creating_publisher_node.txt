ROS2 Publisher Node in Python (Simple Notes)

What is a Publisher?

A publisher is a node that sends data to a topic.
Other nodes (subscribers) can receive this data.

Example in turtlesim:

Publisher Node → /turtle1/cmd_vel → Turtle moves

The publisher sends velocity commands to the turtle.

Example Idea (Draw Circle in Turtlesim)

We send velocity commands continuously.

Linear velocity → forward motion
Angular velocity → turning motion

When both are non-zero, the turtle moves in a circle.

Example concept:

linear.x = 2.0
angular.z = 1.0

Result → Turtle moves in a circle.

Summary

Publisher node sends messages to a topic.
Subscribers receive the data and perform actions.

Publisher → Topic → Subscriber
